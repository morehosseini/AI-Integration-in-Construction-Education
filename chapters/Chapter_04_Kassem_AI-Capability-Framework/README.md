# Chapter 4 — When Technical Outputs No Longer Demonstrate Competence: A Capability Framework for AI-Enabled Built Environment Education

**Authors:** Mohamad Kassem, Miroslaw J. Skibniewski, Abdulmalik Badamasi, M. Reza Hosseini
**Status:** Accepted — final manuscript received 21-Jul-2026

## About this chapter

This chapter presents a capability framework for AI-enabled built environment education, addressing the shift from technical task output to professional competence and non-delegable professional judgment.
